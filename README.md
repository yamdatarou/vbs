### シート構成
```
シート: {
            検索: 学校名
            学校一覧: 学校名, 偏差値
            生徒一覧: 生徒名, 偏差値
            新規書き出し
            学校名, 生徒名
        }
```
###  シーケンス
```
学校一覧を回す
    if 学校一覧.偏差値 === 生徒一覧.偏差値
        新規.学校名
        新規.生徒名
```
